## 🎯 核心特性

### 1. 双端分离设计
- **管理端** (`/admin/*`)：管理员和宿管使用
- **学生端** (`/student/*`)：学生使用
- 登录后根据用户类型自动跳转

### 2. 完善的权限控制
- **路由级别**：meta.roles 配置
- **前端守卫**：router.beforeEach 验证
- **后端验证**：API请求携带Token

### 3. 统一的API管理
- 所有API集中在 `src/api/` 目录
- 统一的请求/响应拦截
- 自动处理Token和错误

### 4. 美观的UI设计
- Element Plus组件库
- 响应式布局
- 渐变色卡片
- ECharts图表

## 🐛 常见问题

### 1. npm install 失败
```bash
npm cache clean --force
npm config set registry https://registry.npmmirror.com
npm install
```

### 2. 页面空白
- 检查后端是否启动
- 检查浏览器控制台错误
- 检查API请求状态

### 3. 登录后跳转错误
- 检查用户类型（userType）
- 查看路由配置
- 确认后端返回数据正确
